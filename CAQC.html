💾 Sessions
Save & Load Sessions
Your work is stored as a JSON file — including all photos. Load it any time to continue.
💾
Save Session
Tap 💾 Save in the top bar — a CAQC_HZ-XXX_2025-xx-xx.json file downloads automatically. This includes all photos.
📂
Open Session
Tap 📂 Open next to Save — select the JSON file you saved earlier to restore everything exactly as you left it.
🔄
Auto-Save
The app silently auto-saves to your browser as you work — so short interruptions won't lose your progress. For long sessions, always Save to a file.
🗑
Clear Session
Open Sessions from the ⚙ menu to start fresh for a new aircraft. Always save first!
💡 Tips
Pro Tips
Get the most out of CA&QC with these workflow tips.
📁
Name Your Photos Well
Name files like SEAT_14A_ARMREST.jpg — the auto-classifier reads the filename to assign the right specialty automatically.
📚
After Library Workflow
Upload all your After photos to the Library at once, then assign them to each finding — much faster than uploading one by one.
🔍
Filter Before Exporting
Use the Status filter to show only OPEN items, then export — gives the team a focused action list.
📱
Works Offline
The app runs entirely in your browser with no internet needed after the first load. Perfect for hangar use.
⚡
Keyboard shortcut: After typing a WO number in the Report table, press Tab to jump to the next field and Enter to confirm edits in text inputs.
